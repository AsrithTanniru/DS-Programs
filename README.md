# DS-Programs

This repository contains a collection of code implementations for various data structure problems. Organized by weeks, each folder focuses on different concepts, making it a valuable reference for students and enthusiasts aiming to strengthen their understanding of data structures.

## 📁 Repository Structure

The repository is divided into weekly folders, each encompassing specific data structure topics:

* `Week-I`
* `Week-III`
* `Week-IV`
* `Week-V`
* `Week-VI`
* `Week-VII`
* `Week-VIII`
* `Week-X`
* `Week-XI`
* `Week-XII`
* `others`

Each folder contains C programs addressing various data structure problems.

## 📥 Cloning the Repository

To clone this repository to your local machine, ensure you have Git installed. Then, execute the following command in your terminal:

```bash
git clone https://github.com/AsrithTanniru/DS-Programs
```

## 📌 Notes

* The programs are primarily written in C.
* Some files may have `.exe` extensions, indicating compiled executables.
* The repository is a work in progress and may receive updates over time.
